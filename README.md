> Crud em node

### Ferramentas 
- Node 
- Express 
- Mongoose 
- MongoDB Atlas 
- Postman

### Objetivos
- Desenvolver aplicação com os métodos do CRUD
- Utilizar o Mongoose para manipulação de itens.
- Debugar os métodos http
- Entender fluxo de dados no back-end de uma API Rest
